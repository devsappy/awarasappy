BUNDLED IMAGES FOLDER
=====================

Place images that will be imported in components here.

How to Use:
-----------
Import images in your React components:

import myImage from './assets/images/my-image.jpg'

function MyComponent() {
  return <img src={myImage} alt="Description" />
}

Benefits:
---------
✓ Optimized by Vite
✓ Hashed filenames (better caching)
✓ Smaller bundle size
✓ TypeScript support

Best For:
---------
- Small to medium images
- Images used in multiple components
- SVG graphics
- Logo images

See IMAGE_GUIDE.md in the root folder for complete documentation!
